通过无线远程使用adb
发表于 2019-03-06| 更新于 2020-07-26|小黑科技
字数总计:231|阅读时长: 1 分钟
|阅读量:

操作步骤

  1. 设置好NDK环境(废话)

  2. 保持手机和电脑在同一局域网内，将数据线连接手机和电脑

  3. 查看当前连接设备情况: adb devices

  4. 查看此时设备的ip地址: adb shell ifconfig (或 adb shell ip addr)

  5. 开启 adb tcp监听: adb tcpip 5555 (端口号自己指定)

  6. 断开数据线，无线连接: adb connect 172.20.19.67:5555

  7. 完成。通过 adb devices 可以看到设备连接已变成无线的方式, 一应操作和线连adb是一样的, 同样也可使用android studio安装app并调试

其他操作

  • 断开连接

  • 解决ubuntu虚拟机无法通过usb连接移动端设备

    在macOS上完成1-5步, ubuntu完成第1步、第6步即可。
